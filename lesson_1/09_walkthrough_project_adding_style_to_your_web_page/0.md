# Applying CSS
## Inline CSS
- Use the `style` attribute to apply some CSS styling to a single element
- All the properties and values of the styles are contained in the single string value for the `style` attribute
- Property-value pairs are separated by `;`
- This mixes the styling information with the content, so it's discouraged

## Internal CSS
- All the CSS styling is prescribed in a `<style>` element in the `<head>` element of the HTML document
- Most convenient to apply CSS styling for a single page

## External CSS
- Include all the CSS code in a separate `.css` file and link it in the HTML document with an `<link>` self-closing tag that contains a `rel="stylesheet"` and a `href` attribute
- This is the preferred way
  - Multiple HTML documents can make use of a CSS file
  - Centralize CSS styling to one file

### Other
- Use HTML tags semantically—each tag names should describe the type of information it is intended to display
- Arguments on whether to use ID selectors in one's CSS aside, use it sparingly
