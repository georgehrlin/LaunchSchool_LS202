### Lengths
Length values come in two different forms: absolute and relative, each of which uses different units of measurement.

#### Absolute Lengths
##### Pixels
The pixel is equal to 1/96th of an inch. Therefore 1 inch = 96 pixels. However, the exact measurement of a pixel varies between high-density and low-density viewing devices.

#### Relative Lengths
Relative lengths are not fixed units of measurement. They rely on the length of another measurement.

##### Percentages
Percentage lengths are defined in relation to the length of another object.

```css
.col {
  width: 50%;
}
```

Here the width of the element with a `col` class attribute value is set to `50%` relative to the width of the element's parent.

##### Em
The em unit is another relative value unit. Its length is calculated based on an element's font size. A single em unit is equivalent to an element's font size.

```css
.banner {
  font-size: 14px;
  width: 5em;
}
```

Here, because the font size and width of elements with a `banner` class attribute is set to 14 pixels and 5 em, the width would equal 70 pixels (14 pixels times 5).

When a font size is not explicitly stated, the em unit is relative to the font size of the closest parent element with a stated font size.

The em unit is often used for styling text, including font sizes and spacing aroun text (like margins and paddings).
