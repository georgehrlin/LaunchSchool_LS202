# Building Your First Web Page
Before we begin building, we need to understand:
- The differences between HTML and CSS
- The syntax of each language
- Some common terminology

## What Are HTML & CSS?
HTML, HyperText Markup Language, gives content structure and meaning by defining the content.
CSS, Cascading Style Sheets, is a presentation language that styles the appearance of content.
The two languages are independent of one another. CSS should not be written inside of an HTML document and vice versa. HTML will always represent content, and CSS will always represent the appearance of the content.

## Understanding Common HTML Terms
#### Elements
Elements are designators that define the structure and content of objects within a page. Elements are defined by the use of less-than and greater-than angle brackets, `< >`, surrounding the element name.

Common examples include headings (`<h1>` through `<h6>`), paragraphs (`<p>`), etc.

##### Self-Closing Elements
Self-closing elements receive their content or behaviour from attributes within a single tag. E.g., `<meta charset="utf-8">`. Other common examples include: `<br>`, `<embed>`, `<hr>`, `<img>`, etc.

#### Tags
A tag is defined by a single use of less-than and greater-than brackets surrounding an element name. Tags usually come in pairs of opening and closing tags. E.g., `<a>` is the opening tag of an anchor link, while `</a>` is the closing tag.

The content that falls between the opening and closing tags is the content of the element.

#### Attributes
Attributes are properties used to provide additional information about an element. Attributes are defined within the opening tag, after an element's name, and generally include a name and a value. For instance, below is an anchor element that includes an `href` attribute:
`<a href="https://launchschool.com/">Launch School</a>`

Examples of common attributes:
- `id`: identifies an element
- `class`: classifes an element
- `src`: specifies a source for embeddable content
- `href`: provides a hyperlink reference for a linked source
