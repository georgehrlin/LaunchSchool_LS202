## Setting Up the HTML Document Structure
All HTML documents have a required structure that includes `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`:
- `<!DOCTYPE html>` informs web browsers the version of HTML being used
- `<html>` marks the beginning of the HTML document
- `<head>` then identifies the top of the document, including any metadata (like document title) with content that is not displayed on the web page itself
- `<body>` is where all of the visible content within the web page goes into

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a web page.</p>
  </body>
</html>
```
