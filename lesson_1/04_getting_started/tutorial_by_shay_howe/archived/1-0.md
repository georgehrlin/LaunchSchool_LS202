## What Are HTML & CSS?
HTML, Hypertext Markup Language, gives content structure and meaning by defining the content as, for example, headings, paragraphs, or images.

CSS, Cascading Style Sheets, is a presentation language created to style the appearance of content using, for example, fonts or colors.

They are independent of one another. CSS should not be written inside of an HTML document and vice versa. As a rule, HTML will always represent content, and CSS will always represent the appearance of that content.

## Understanding Common HTML Terms
### Elements
Elements are designators that define the structure and content of objects within a page. Common examples include multiple levels of headings (the `<h1>` through `<h6>` elements), paragraphs (the `<p>` element), `<a>`, `<div>`, etc.

Elements are identified by the use of less-than and greater-than angle brackets (`< >`) surrounding the element name.

### Tags
The use of less-than and greater-than angle brackets surrounding an element creates a tag. Tags most commonly occur in pairs of opening and closing tags.

An opening tag marks the beginning of an element. It consists of a less-than sign followed by an element's name, and then ends with a greater-than sign; for example, `<div>`.

A closing tag marks the end of an element. It consists of a less-than sign followed by a forward slash and the element's name, and ends with a greater-than sign; for example `</div>`.

The content that falls between the opening and closing tags is the content of that element.

### Attributes
Attributes are properties used to provide additional information about an element.

The most common attributes include:
- The `id` attribute, which identifies an element
- The `class` attribute, which classifies an element 
- The `src` attribute, which specifies a source for embeddable content
- The `href` attribute, which provides a hyperlink reference to a linked resource

Attributes are defined within the opening tag, after an element's name. Generally attributes include a name and a value. The format for these attributes consists of the attribute name followed by an equals sign and then a quoted attribute value. For example:
```html
<a href="http://launchschool.com/">Launch School</a>
```
The preceding code displays the text "Launch School" on the web page and takes users to `http://launchschool.com` when clicked:
- The anchor element is declared with the opening `<a>` and closing `</a>` tags encompassing the text
- The hyperlink reference attribute and value are declared with `href="http://launchschool.com"` in the opening tag

## Setting Up the HTML Document Structure
All HTML documents have a required structure that includes the following declaration and elements: `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`.

`<!DOCTYPE html>`, the document type declaration, informs web browsers the version of HTML being used and is placed at the very beginning of the HTML document. `<!DOCTYPE html>` deisgnates the latest version of HTML.

Inside the `<html>` element, the `<head>` element identifies the top of the document, including any metadata. The content inside the `<head>` element is not displayed on the web page itself, though it may include the document title (which is displayed on the title bar in the browser window), links to any external files, or any other beneficial metadata.

All of the visible content within the web page falls within the `<body>` element.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a web page.</p>
  </body>
</html>
```

#### Self-Closing Elements
Self-closing elements only receive their content or behaviour from attributes within a single tag (like the `<meta>` element). Other common self-closing elements include:
- `<br>`
- `<embed>`
- `<hr>`
- `<img>`
- `<input>`
- `<link>`
- `<meta>`
- `<param>`
- `<source>`
- `<wbr>`
