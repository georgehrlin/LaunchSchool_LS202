`<p>` is interpreted as regular HTML code by the browser. To "escape" it, replace the left and right angle brackets with character entities: `&lt;` and `&gt;`, respectively.

`&` is handled by HTML5 as a regular character in most cases, but if that `&` includes some non-whitespace characters between it and the next semicolon `;`, the browser may interpret it as a character entity. Therefore, `&amp;` is often deliberately used just in case.

Use `&quot;` to embed a double quote inside a double-quoted attribute value:
```html
<img src="hello-world.jpg" alt="Greetings: &quot;Hello, World&quot;">
```
It's common practice to replace `"` with `&quot;` everywhere because it's usually impossible to know which strings will need `&quot;`.

The format of all character entities is an `&` followed by one or more alphanumeric characters and then terminated with a `;`. All of then have numeric versions with most having alphanumeric names. Use the alphanumeric name whenever possible.
