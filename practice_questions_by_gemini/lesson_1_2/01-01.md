# 4
The Specificity Race
Given the following HTML and CSS, what color will the text "Hello World" be?
```html
<p id="intro" class="highlight">Hello World</p>
```
```css
p { color: red; }
#intro { color: blue; }
.highlight { color: green; }
```

A:
In the CSS code there are three selectors of various kinds. ID selectors always have the highest specificity weight. Therefore, `#intro { color: blue; }` will win out among the three selectors and `Hello World` will be blue.

# 7
Total Width Calculation (Content-Box)
An element has the following CSS:
```css
div {
  box-sizing: content-box;
  width: 400px;
  padding: 20px 10px;
  border: 5px solid black;
  margin: 15px;
}
```
What is the total horizontal space (from the left margin edge to the right margin edge) this element occupies in the layout?

A:
Because of `box-sizing: content-box;`, `width: 400px;` only applies to `<div>`'s content area.
Therefore, the total horizontal space = 400px + 10px (left padding) + 10px (right padding) + 5px (left border) + 5px (right border) + 15px (left margin) + 15px (right margin) = 460px.
To be completely honest, I am not sure if the values of `margin` should be included in the calculation above. I did so because the question asks for "total horizontal space".

**Correction:**
In the Box Model, "Total Width" usually refers to the space the box takes up including borders and padding. Margins, on the other hand, are technically the space around the box, not part of the box's size itself. Therefore, the total width is 430px.

# 9
Display Behaviors
Briefly explain the layout difference between these three scenarios:
1. Two `div` elements with `display: block`.
2. Two `span` elements with `display: inline`.
3. Two `div` elements with `display: inline-block` and a width of `45%`.

A:
1. The first `div` element will be rendered on top of the second `div` because `block` elements automatically take up the entire horizontal space of their container.
2. The two `span` elements will be displayed on the same line and right next to each other.
3. The two `div` elements will be displayed on the same line and right next to each other because the sum of their widths is less than the width of their container.

# 10
Centering Challenge
You have an image that is 300px wide inside a `section`. Write the CSS required to:
1. Make the image behave like a block element.
2. Center it horizontally within the section.
3. Ensure that if the screen gets narrower than 300px, the image doesn't overflow (it should shrink instead).

A:
```css
img {
  display: block;
  margin-left: 100%;
  margin-right: 100%;
}
```
I don't know how to achieve 3.

**Correction:**
```css
img {
  display: block;
  margin: 0 auto;
  max-width: 100%;
  width: 300px;
}
```
